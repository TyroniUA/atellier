# Atelier Landing — Checkpoint One

A lightweight Next.js landing page prototype for a boutique AI / cybersecurity agency focused on trust, responsibility, and long-term relationships.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- Built with Next.js App Router
- Styled with Tailwind CSS
- Animations with Framer Motion
- Minimal local Button component included
